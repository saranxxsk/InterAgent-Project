# InterAgent

A REST-Based Messaging Protocol for Agentic AI Orchestration.

## Features
- Multi-agent communication using REST APIs
- Dynamic task delegation
- Capability-based agent discovery
- Fault-tolerant orchestration
- Scalable architecture

## Technologies
- Python
- Django
- Django REST Framework
- SQLite
- REST APIs

## Project Structure
interagent/
interagent_project/
manage.py

## Installation

```bash
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```
